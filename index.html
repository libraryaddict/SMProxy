SMProxy

A Minecraft proxy for software development

New Snapshot: SMProxy for 13w03a can be downloaded here.

SMProxy is a Minecraft proxy targeted at software development with the Minecraft protocol, based on the powerful Craft.Net library. It allows you to log network traffic between a Minecraft client and server, as well as providing a good base for any proxy-related project with a solid plugin API that allows you to have complete control over any Minecraft connection.
